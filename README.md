# 交易陪练AI - MVP原型

一个帮助交易新手建立交易纪律的Web应用，通过纸盘交易和AI评分系统，在零风险环境下学习交易。

## 🎯 核心功能

### 1. **纸盘交易**
- 虚拟$500初始资金
- 接入真实BTC价格（CoinGecko API）
- 支持做多/做空
- 杠杆 1-10x
- 止损/止盈设置
- 实时盈亏计算

### 2. **AI教练评分**
- **入场评分**：检查止损、仓位大小、杠杆是否合理
- **出场评分**：判断是否按计划执行、有无情绪化交易
- **综合建议**：胜率、盈亏比、最大回撤分析

### 3. **复盘日志**
- 查看所有历史交易
- AI评分和反馈记录
- 交易统计（胜率、总盈亏、回撤）

### 4. **风险管理**
- 自动触发止损/止盈
- 仓位大小警告
- 高杠杆风险提示

## 🏗️ 技术栈

- **前端框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **价格数据**: CoinGecko API (免费)
- **数据存储**: LocalStorage (JSON)

## 🤖 AI评分规则 (V1)

### 入场评分 (0-100分)
- ❌ **未设止损**: -30分
- ⚠️ **未设止盈**: -10分
- ❌ **仓位过大** (>50%): -30分
- ⚠️ **仓位较大** (20-50%): -10分
- ⚠️ **高杠杆** (>5x): -20分
- ❌ **风险回报比差** (<1:1): -20分

### 出场评分 (0-100分)
- ⚠️ **未按计划出场**: -20分
- ⚠️ **提前止盈**: -15分
- ❌ **情绪化交易** (持仓<1h): -25分
- ❌ **未设止损且亏损>20%**: -30分

## ⚠️ 重要说明

1. **仅供学习**：本应用仅用于教育目的，不构成投资建议
2. **数据存储**：使用浏览器LocalStorage，清除缓存会丢失数据
3. **价格数据**：CoinGecko免费API有请求限制，如触发限制会使用模拟数据
4. **V1版本**：AI评分基于规则，未接入真实LLM
