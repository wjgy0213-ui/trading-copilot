## ⚠️ 注意事项

1. **CoinGecko API 限制**
   - 免费版每分钟50次请求
   - 已实现fallback机制（模拟数据）
   - 生产环境建议升级API计划

2. **数据持久化**
   - 使用 LocalStorage 存储
   - 清除浏览器缓存会丢失数据
   - V2 可考虑使用 Supabase

3. **浏览器兼容性**
   - 需要现代浏览器支持
   - 建议 Chrome/Firefox/Safari 最新版
